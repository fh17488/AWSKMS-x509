# AWSKMS-x509

This repository can be used to create a public x509 certificate using the AWS KMS asymmetric key.

## Maven commands
# Install dependencies
mvn clean install
# Package into JAR
mvn package
# Fetch classpath
mvn dependency:build-classpath
